# Tic Tac Toe Game
### Based on JavaFX

JavaFX is a software platform for creating and delivering desktop applications, as well as rich web applications that can run across a wide variety of devices. 

# Test
Below are the steps to run the game
- ## Clean install
    ```
    mvn clean install
    ```
- ## Compile
    ```
    mvn compile
    ```
- ## Run
    ```
    mvn javafx:run
    ```
